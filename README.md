random-data-generator-mongo
===========================

Generates and inserts random data into mongodb

You configuartion
-----------------
Add a file named config.json in the project root with the following contents. Replace with your
details.

    {
      "db": {
        "host": "x.x.x.x",
        "port": "27017",
        "user": "user",
        "pass": "password",
        "db": "test"
      }
    }
